## ファイル構成

```
docs/
```

## AI駆動での活用方法

### 1. 戦略検討・ブレスト時
```
「04_marketing-strategy.md を参照して、
 リード獲得を増やすための新しいアイデアを3つ提案してください」
```

### 2. コンテンツ制作時
```
「01_product-overview.md と 02_target-personas.md を参照して、
 ペルソナAに響くブログ記事のタイトル案を5つ作成してください」
```

### 3. 広告文作成時
```
「03_usp-competitive-advantage.md のUSPを参照して、
 Google検索広告の見出しを10パターン作成してください」
```

### 4. レポート作成時
```
「06_kpi-metrics.md のテンプレートを使って、
 以下のデータで週次レポートを作成してください」
```

### 5. 施策の振り返り時
```
「05_content-campaigns.md のキャンペーン計画と実績を比較して、
 改善点を分析してください」
```

## クイックリファレンス

### サービス概要（1行）
> 介護・在宅医療向けAI人材育成オンラインスクール（助成金で最大75%オフ）

### 主なUSP
1. 介護業界特化
2. 伴走型サポート（学習→業務改善まで）
3. 助成金対応（実質月5,833円〜）
4. 未経験者95%対応
5. 堀江貴文監修

### ターゲット
- 介護施設の管理者・経営者
- 訪問介護事業所の事務担当
- 社会福祉法人の研修担当

### 主要数値
- 導入企業: 200社以上
- 受講者: 2,000人以上
- 講座数: 240以上
- 年間料金: 31万円/人（助成金後7万円〜）
