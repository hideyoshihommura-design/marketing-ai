# マーケティング戦略フレームワーク

## 戦略概要

### ミッション
介護・在宅医療業界にAI活用人材を育成し、業務効率化と人材不足解消に貢献する

### ビジョン
介護業界のDX推進における第一想起ブランドになる

### 戦略目標
- [ ] 導入企業数: 200社 → 500社（2.5倍）
- [ ] 受講者数: 2,000人 → 5,000人（2.5倍）
- [ ] 認知度: 介護事業者の〇〇%が認知

## セグメント別マーケティング施策

### 小規模事業所（〜30名）向け

**メッセージ:**
> 「2名から始められる。助成金で月5,833円から。」

**施策:**
- [ ] Google検索広告（「介護 AI 研修」「介護 業務効率化」）
- [ ] Facebook広告（介護経営者向けターゲティング）
- [ ] 導入事例（同規模事業所の成功事例）
- [ ] 無料相談会（オンライン）

### 中規模法人（30-100名）向け

**メッセージ:**
> 「組織的なAI人材育成で、競合との差別化を。」

**施策:**
- [ ] 介護業界展示会への出展
- [ ] セミナー開催（DX推進・助成金活用）
- [ ] 社内稟議用資料の提供
- [ ] 経営層向けプレゼンテーション

### 大規模法人（100名〜）向け

**メッセージ:**
> 「全社員のAIリテラシー向上を、一括でサポート。」

**施策:**
- [ ] アカウントベースドマーケティング（ABM）
- [ ] カスタマイズ研修提案
- [ ] 経営者ネットワークでのリファラル
- [ ] 大規模割引・ボリュームディスカウント提案

## 価格戦略

### 料金体系

| 人数 | 料金（税抜） | 助成金75% | 実質負担 | 月額換算/人 |
|------|------------|----------|----------|------------|
| 2人 | 62万円 | 48万円 | 14万円 | 5,833円 |
| 4人 | 124万円 | 96万円 | 28万円 | 5,833円 |
| 10人 | 310万円 | 240万円 | 70万円 | 5,833円 |
| 14人 | 400万円 | 330万円 | 70万円 | 4,166円 |

### 価格訴求のポイント

1. **助成金後の実質価格で訴求**
   - 「月額5,833円〜」を前面に

2. **4名以上のメリット強調**
   - 社労士サポートフィー無料

3. **大規模導入の割引**
   - 14名以上でさらにお得
