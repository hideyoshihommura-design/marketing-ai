# ターゲット顧客・ペルソナ定義

## 優先ペルソナ：介護法人経営者

### 基本プロファイル

| 項目 | 内容 |
|------|------|
| 名前 | 佐藤 健太（仮名） |
| 年齢 | 45歳 |
| 役職 | 社会福祉法人 理事長 / 株式会社 代表取締役 |
| 事業規模 | 1〜3施設、従業員30〜100名程度 |
| 年商 | 2〜10億円 |
| 家族 | 既婚、子供2人（中学生・高校生） |
| 学歴 | 大卒（福祉系 or 経営系） |
| 経歴 | 親から事業承継 or 介護業界で独立 |

### Pain（痛み）詳細

#### 最大の痛み：人手不足・採用難
```
「求人出しても応募が来ない」
「やっと採用しても3ヶ月で辞める」
「残った職員に負担が集中して悪循環」
「人がいないから新しいことができない」
```

#### 関連する痛み
| 痛み | 具体的な状況 |
|------|-------------|
| 事務作業の非効率 | FAX、手書き、Excel手入力が山ほどある |
| 管理者の疲弊 | 現場も管理もやって燃え尽きそう |
| 採用コストの増加 | 紹介料で1人50万円、でも辞める |
| 経営の将来不安 | このままで5年後やっていけるのか |

---

### Gain（理想の状態）

```
「少ない人数でも回る仕組みを作りたい」
「職員がやりがいを持って働ける環境にしたい」
「他施設より一歩先を行きたい」
「経営者として成長し続けたい」
```

---

### 導入障壁（なぜ動けないのか）

#### 最大の障壁：何から始めればいいか分からない

```
「AIって言われても具体的に何するの？」
「うちの業務のどこに使えるの？」
「情報が多すぎて整理できない」
「相談できる人がいない」
```

#### その他の障壁
| 障壁 | 心の声 |
|------|--------|
| 費用への不安 | 「研修に何十万もかけて元取れる？」 |
| 現場への不安 | 「うちの職員、PC苦手な人多いけど大丈夫？」 |
| 時間の確保 | 「忙しくて研修受ける時間なんてない」 |
| 効果への疑問 | 「本当に業務改善につながるの？」 |

---

### 購買トリガー（何がきっかけで動くか）

#### 最強のトリガー：同業の経営者が導入したと聞いた

```
「あの○○さんのところ、AI研修始めたらしいよ」
「え、そうなの？どこの？」
「ホリエモンAI学校ってやつ」
「...ちょっと調べてみるか」
```

**→ 同業者の動きが最大の行動トリガー**

#### その他のトリガー
| トリガー | 状況 |
|----------|------|
| 助成金の存在を知る | 「75%オフならリスクないじゃん」 |
| 具体的な成果事例 | 「報告書作成が半分の時間に？」 |
| ホリエモンの発信を見る | 「やっぱりAIやらないとダメか」 |
| 年度末・予算消化 | 「助成金使えるうちに」 |

### 意思決定プロセス

```
1. 認知
   「ホリエモンAI学校ってあるんだ」（SNS、口コミ）
   ↓
2. 興味
   「介護向けなんだ、ちょっと見てみるか」（LP訪問）
   ↓
3. 検索
   「評判どうなの？」「助成金って本当？」（Google検索）
   ↓
4. 比較検討
   「他にもあるのかな」「ここが良さそうかも」
   ↓
5. 相談
   「まず話聞いてみるか」（無料相談）
   ↓
6. 確認
   「助成金ちゃんと使える？」（社労士面談）
   ↓
7. 決定
   「よし、やろう」「○人で申し込もう」
```

### 広告・LPで刺さるメッセージ

#### ヘッドライン案

**痛みに刺す（問題提起）:**
```
「人が足りない」を、AIで解決しませんか？
```

**障壁を取り除く:**
```
「何から始めれば...」その悩み、私たちが解決します
```

**トリガーを活用:**
```
すでに200社が導入。あなたの施設は？
```

**価格障壁を消す:**
```
助成金で実質75%オフ。月5,833円でAI人材育成
```

#### サブコピー案
```
・介護業界特化だから、学んだその日から使える
・PC操作ができれば大丈夫。受講者の95%が未経験スタート
・動画を見て終わりじゃない。業務改善まで伴走します
・堀江貴文監修。本気の経営者のためのAI研修
```

#### CTA（行動喚起）案
```
・まずは無料で相談する
・助成金が使えるか確認する
・導入事例を見る
・資料をダウンロードする
```

---

### LP構成案（経営者向け）

```
1. ファーストビュー
   「人が足りない」をAIで解決しませんか？
   ↓
2. 共感セクション
   こんなお悩みありませんか？
   - 採用しても人が来ない
   - 事務作業に追われて本業に集中できない
   - AIを導入したいけど何から始めれば...
   ↓
3. 解決策提示
   ホリエモンAI学校 介護校なら、
   「何から始めるか」から「業務改善」まで伴走
   ↓
4. 実績・信頼
   導入企業200社以上、受講者2,000人以上
   業界No.1の評価（分かりやすさ・満足度）
   ↓
5. 導入事例
   「報告書作成が半分の時間に」○○施設様
   ↓
6. 料金・助成金
   年間31万円 → 助成金で実質7万円
   月額5,833円でAI人材育成
   ↓
7. 不安解消
   Q. うちの職員でもできる？
   A. 95%が未経験からスタート
   ↓
8. CTA
   まずは無料で相談する
```

## セカンダリペルソナ

### ペルソナB: 施設長・管理者（情報収集者）

経営者から「調べておいて」と言われる立場。
実際にサービスを調査し、上に報告する役割。

| 項目 | 内容 |
|------|------|
| 年齢 | 35-50歳 |
| 役職 | 施設長、管理者、事務長 |
| 特徴 | 現場も分かり、経営も理解している |
| 痛み | 経営者と現場の板挟み |
| 動機 | 自分の提案で会社を変えたい |

**この人向けのメッセージ:**
```
「社長に提案できる資料、揃っています」
「稟議が通る助成金活用プラン」
```

### ペルソナC: 事務担当（実受講者）

実際に研修を受ける立場。日々の業務効率化に直結。

| 項目 | 内容 |
|------|------|
| 年齢 | 25-45歳 |
| 役職 | 事務、介護職員 |
| 特徴 | ITに苦手意識があることも |
| 痛み | 毎日の事務作業が大変 |
| 動機 | 楽になりたい、スキルアップしたい |

**この人向けのメッセージ:**
```
「毎日の報告書作成、もっと楽にしませんか？」
「PC操作ができれば大丈夫」
```

## ペルソナ活用チェックリスト

### 広告制作時
- [ ] ヘッドラインは「痛み」に刺さっているか
- [ ] 「何から始めれば」の障壁を取り除いているか
- [ ] 助成金の価格メリットは明示されているか
- [ ] 同業者の実績・事例は入っているか

### LP制作時
- [ ] 経営者の言葉で書かれているか
- [ ] 不安を先回りして解消しているか
- [ ] CTAは明確か
- [ ] スマホで見やすいか（SNS流入を想定）

### 営業トーク設計時
- [ ] 人手不足の悩みからヒアリングを始めているか
- [ ] 同業他社の事例を用意しているか
- [ ] 助成金の説明は簡潔にできるか
