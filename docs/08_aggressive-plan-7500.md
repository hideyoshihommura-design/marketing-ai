# ホリエモンAI学校 介護校 積極的プラン（Year 1: 7,500万円目標）（超シビア想定）

## 概要

本ドキュメントは、Year 1からFC売上7,500万円を目指す積極的プランです。
**全転換率を超シビア（保守的）に設定**し、最悪ケースでも目標達成可能な計画としています。

- **作成日**: 2025年1月
- **対象期間**: Year 1
- **目標FC売上**: 7,500万円
- **必要条件**: 予算倍増、広告強化
- **想定**: 超シビア想定（全CVRを保守的に設定）

### 結論：必要投資と期待リターン（超シビア想定）

| 項目 | 数値 |
|------|------|
| **年間マーケティング予算** | **3,120万円**（月260万円） |
| **その他経費** | **300万円**（内訳は下記参照） |
| **総投資額** | **3,420万円** |

| **粗利（売上-マーケ費）** | **4,380万円** |
| **営業利益（売上-総投資額）** | **4,080万円** |
| **利益率** | **54%** |
| **ROAS（対マーケ費）** | **240%** |
| **CPA（マーケ費÷契約件数）** | **25.8万円** |
| **CAC（総投資額÷契約件数）** | **28.3万円** |

> **CPA vs CAC**: CPAはマーケティング費用のみ÷契約件数。CACは総投資額（マーケ費+その他経費）÷契約件数。CACが実質的な1件あたりの顧客獲得コスト。
>
> **投資判断**: LTV/CAC = 62万÷28.3万 = **2.19倍**（超シビア想定のため健全水準3倍を下回るが、十分に投資回収可能）。

### 広告予算ガイダンス

**いくらかければいいのか（推奨予算）**

目標7,500万円を達成するために、超シビアな転換率で逆算した必要予算です。

| レベル | 月間予算 | 年間予算 | ROAS | 意味 |
|--------|---------|---------|------|------|
| **推奨予算（目標達成）** | **260万円** | **3,120万円** | **240%** | **超シビア想定で121件を獲得するために必要な投資額** |
| **最大許容（ROAS 200%）** | **313万円** | **3,750万円** | **200%** | **投資の2倍は回収できる上限ライン** |
| 損益分岐 | 600万円 | 7,200万円 | 104% | これ以上かけると赤字（その他経費300万含む） |

> **結論**: 月260万円（年3,120万円）を推奨。最大でも月313万円（年3,750万円）を超えないこと。損益分岐は月600万円だが、利益が残らないため現実的ではない。

## 10. 超シビア想定の転換率一覧

本プランで採用している転換率は、すべて保守的（超シビア）な数値です。

| 転換ポイント | 超シビア想定 | 一般的な想定 | 差分 |
