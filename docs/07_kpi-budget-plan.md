# ホリエモンAI学校 介護校 KPI・予算計画書（堅実プラン）

## 概要

本ドキュメントは、ホリエモンAI学校介護校（あおぞらケアグループ運営）のマーケティングKPI設定および予算計画を定めたものです。**すべての前提を厳しく設定した堅実プラン**です。

- **作成日**: 2025年1月
- **対象期間**: Year 1〜Year 2
- **性質**: Year 1は投資・検証期間と位置づけ

---

## エグゼクティブサマリー

### 結論：Year 1は「投資期間」である

| 項目 | Year 1 | Year 2 |
|------|--------|--------|
| **マーケティング予算** | **720万円**（月60万円） | **960万円**（月80万円） |
| **その他経費** | **200万円** | **250万円** |

| **契約件数** | **16件** | **40件** |

> **投資判断**: Year 1は赤字〜収支トントンを覚悟する。この期間はファネル構築・データ蓄積・導入事例作成のための「投資」である。Year 2以降に回収する設計。Q2終了時点で継続・撤退の判断を行う。

---

### なぜ「赤字覚悟」なのか

```
【Year 1 の現実】

新規参入 → ブランド認知ゼロ
         → 導入事例ゼロ
         → LP・広告の勝ちパターン未確立
         → ウェビナー運営ノウハウなし
         → 営業トーク・提案資料の改善途上

        ↓ これらすべてが影響

  ・CVRは業界ベンチマーク下限以下で推移
  ・CPAは40万円超（売上の85%相当）
  ・Q1-Q2はほぼ赤字

        ↓ しかし

  ・Q3以降にデータ蓄積・事例完成でCVR改善
  ・Year 2でCPA 24万円まで改善を見込む
  ・Year 2のROAS 229%で投資回収フェーズへ
```

## 3. 目標設定の根拠

### 3.1 市場規模

| 指標 | 数値 | 出典 |
|------|------|------|
| 介護事業所数（全国） | 約21万事業所 | 厚生労働省 |
| 法人数（複数事業所運営含む） | 約8万法人 | 推計 |
| DX投資意向のある法人 | 約30% | 業界調査 |
| **潜在ターゲット** | **約2.4万法人** | 推計 |

**Year 1目標: 16件 = 潜在ターゲットの0.07%**

> この浸透率は新規参入として現実的。ただし「DX投資意向30%」自体が楽観的な可能性もある。「AI研修にまで予算を割ける法人」はさらに少ない。

### 3.2 転換率（CVR）根拠

**すべてのCVRを業界ベンチマーク下限で設定。一部はベンチマーク下限をさらに下回る水準。**

| 転換ポイント | 本計画 | 業界ベンチマーク | 設定理由 |
|-------------|--------|-----------------|---------|
| 訪問 → ウェビナーLP訪問 | **20%** | 20-40% | ベンチマーク下限。導線未整備 |
| ウェビナーLP → 申込 | **4%** | 5-12% | **ベンチマーク下限以下**。無名ブランド |
| 申込 → 参加 | **40%** | 40-60% | ベンチマーク下限。リマインド体制未整備 |
| 参加 → 商談希望 | **10%** | 10-20% | ベンチマーク下限。コンテンツ品質未成熟 |
| 訪問 → 資料請求 | **1.5%** | 2-5% | ベンチマーク下限以下。資料の信頼性不足 |
| 資料請求 → 商談希望 | **5%** | 5-12% | ベンチマーク下限 |
| 訪問 → LINE登録 | **2%** | 2-5% | ベンチマーク下限 |
| LINE登録 → 商談希望 | **2%** | 2-5% | ベンチマーク下限 |
| 商談設定 → 実施 | **65%** | 65-80% | ベンチマーク下限。日程調整の歩留まり |
| 商談 → 成約 | **18%** | 15-30% | ベンチマーク下限寄り。新規商材・実績なし |

### 3.3 ファネル計算（H2月間目標）

```
サイト訪問: 1,800 ························· 100%
  │
  ├─【ウェビナー経由】
  │   ウェビナーLP訪問: 360（20%）
  │   申込フォーム送信: 14（4%）
  │   参加: 6（40%）
  │   商談希望: 0.6件（10%）
  │
  ├─【資料請求経由】
  │   資料請求フォーム送信: 27（1.5%）
  │   商談希望: 1.4件（5%）
  │
  └─【LINE登録経由】
      LINE登録: 36（2%）
      商談希望: 0.7件（2%）

  ─────────────────────────────────────
  商談設定（マーケ経由）: 2.7件 ············ 0.15%
  + 紹介・パートナー経由: 1.0件
  商談設定（合計）: 3.7件
    ↓ 実施率 65%
  商談実施: 2.4件
    ↓ 成約率 18%
  成約（マーケ経由）: 0.3件
  + 紹介経由成約（成約率30%）: 0.3件
  ─────────────────────────────────────
  **合計成約: 0.6件/月（H2初期）**

  ※ Q4にはCVR改善+紹介増で1.5〜2件/月を目指す
```

> **全体CVR（訪問→成約）: 0.03%（マーケ単体）**。紹介含めても0.05%以下。法人B2Bの全体CVRは0.1〜0.5%が一般的だが、新規参入Year 1としてはこの水準から開始し、Year 2で0.15%以上を目指す。

### 3.4 なぜ月間成約0.6件なのに年間16件なのか

```
Q1（月間0.1件水準）: 仕組み構築期 → 1件
Q2（月間0.5件水準）: 初期最適化 → 3件
Q3（月間0.8件水準）: CVR改善開始 → 5件
Q4（月間1.5件水準）: 事例蓄積・紹介開始 → 7件
─────────────────────────────────
合計: 16件
```

> Q3-Q4で急増するのは、導入事例の完成（Q2末）、ウェビナーコンテンツの成熟、広告最適化の効果が出始めるため。ただし、この改善カーブ自体が楽観的な可能性もある。

## 9. リスクと対策

### 9.1 リスク一覧

| リスク | 影響度 | 発生確率 | 対策 |
|--------|--------|---------|------|
| Q2まで成約ゼロ | **致命的** | **中（30%）** | 紹介・パートナーチャネルを並行で開拓 |
| CPAが60万円超に高騰 | 高 | 中（30%） | 効率の悪いチャネル停止、予算縮小 |
| 平均2名契約が大半 | 高 | 高（50%） | 3名以上の価格インセンティブ設計 |
| 検索ボリューム不足 | 中 | 中（40%） | Meta/LINE中心の配分に変更 |
| ウェビナー申込が集まらない | 高 | 高（40%） | テーマ変更、広告強化、個別相談会へ移行 |
| 営業リソース不足 | 高 | 中（30%） | 早期に兼任体制でも営業担当を確保 |
| 商談成約率10%以下 | 高 | 低（20%） | 提案資料改善、助成金サポート強化 |

### 9.2 撤退・縮小判断基準

| 時点 | 条件 | アクション |
|------|------|----------|
| **Q2終了時** | 累計成約2件以下かつCPA 80万円超 | 予算を月30万円に縮小 |
| **Q3終了時** | 累計成約5件以下 | 広告費ゼロ、紹介のみに切替 |
| **Year 1終了時** | 累計成約8件以下 | マーケ事業の撤退を検討 |

> **撤退 ≠ 失敗**: 早期にデータで判断し、損失を最小化することが重要。「あと少しやれば...」で赤字を拡大しないこと。
