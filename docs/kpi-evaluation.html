Critical Issue

致命的な問題：事業モデルの誤認

ChatGPTは「介護施設の入居者獲得」のKPIを設計しましたが、実際の事業は介護事業者向けAI研修サービスの販売です。売るものが違えばKPIはすべて変わります。

ChatGPTの認識

介護施設を運営して入居者を集める事業
KPI例：入居率、見学→入居転換率、求人応募数

実際の事業モデル

介護事業者にAI研修を販売するBtoB/BtoC事業
KPI例：LINE登録数、面談予約数、契約転換率、LTV

Budget Model

予算逆算モデル

ChatGPTの予算案の問題

Google広告 月10万円 → 根拠なし
地域SNS広告 月5万円 → オンライン事業に地域限定不要
求人広告 月10万円 → 集客でなく採用の話
問合せCPA 5,000円 → BtoB研修では非現実的
時給500円で削減効果計算 → 最低賃金以下

正しいアプローチ

売上目標から必要リード数を逆算
→ 業界ベンチマークCPAを適用
→ 必要広告予算を算出
→ LTV/CACで投資効率を検証

逆算シミュレーション（月間売上目標 300万円の場合）

法人売上目標186万円
内訳: 2社 x 3名 x 31万円
個人売上目標114万円
内訳: 7名 x 16.28万円（入学金）
必要法人リード数27件/月
面談転換率 30% → 成約率 25%
法人リードCPA（業界相場）30,000円
必要広告予算（法人のみ）810,000円/月
ChatGPTの「CPA 5,000円」との差異：BtoB研修サービスのリード獲得CPAは一般的に15,000〜50,000円。5,000円は消費者向けECの水準であり、このビジネスでは非現実的です。
